Primero que nada se tiene que abrir el scrip de mysql, importalo en MySqlWorkbench y ejecutar todo el scrip, esto va a crear la base de datos con las tablas asignadas en el backend.
Para poder ejectuar la API rest, se debe abrir la carpeta backend con el VS code, seguido de eso se debe abrir una terminal e ingresar el comando "node app.js" para poder proseguir con postman
Al iniciar postman se deben probar los endpoints, selecciona el json que viene en la carpeta raiz y arrastralo en el apartado izquierdo, esto importara los test de Postman.
Finalmente para que deje de estar activa la api se hace ctrl + c en la terminal.
